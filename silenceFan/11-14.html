2014年11月14日

  1. Mybatis配置文件问题:
    Mybatis的配置文件中，配置数据库参数标签中存在一个id用来表示当前的数据库,如果数据库有多个标签也可以随着数据库增加，在标签有一个default属性表示默认使用哪个数据库参数，给定一个标签的id值就行。 
    <?xml version="1.0" encoding="UTF-8"?>
    <!DOCTYPE configuration PUBLIC "-//mybatis.org//DTD Config 3.0//EN" "http://mybatis.org/dtd/mybatis-3-config.dtd">
    <configuration>
    	<environments default="account">
    		<environment id="account">
    			...
    		</environment>
    	</environments>
    	<mappers>
    		...
    	</mappers>
    </configuration>

    在SqlSessionFactory的生成类中，调用SqlSessionFactoryBuilder()的build方法生成session工厂，在build方法中有两个参数，第一个指定Mybatis的配置文件Config.xml的路径，另一个为Mybatis的配置文件中数据库参数的id值，可以再这里指定一个要使用的数据库参数。
  2. 使用MD5对字符串进行加密:
    添加POM中的JAR包 
    		<dependency>
      			<groupId>cn.gyyx.core</groupId>
      			utility
      			<version>0.1</version>
    		</dependency>

    使用MD5.encode(String md);方法可以将字符串md进行MD5加密。

  4. SQL Server的登录问题:
    由于需要用到Mybatis连接数据库，在Mybatis配置文件Config.xml中需要配置连接数据库的username和password，所以不能使用Windows身份验证登录数据库，需要新建一个登录名，强制使用密码，并为该登录名授予连接数据库和登录权限，在Windows身份验证连接的服务器的属性中，设置SQL Server和Windows身份验证模式，接着重复数据库服务，就可以使用登录名正常登录了。
  5. 什么是Velocity:
    Velocity是一个基于java的模板引擎（template engine）。它允许任何人仅仅简单的使用模板语言（template language）来引用由java代码定义的对象。 当Velocity应用于web开发时，界面设计人员可以和java程序开发人员同步开发一个遵循MVC架构的web站点，也就是说，页面设计人员可以只关注页面的显示效果，而由java程序开发人员关注业务逻辑编码。
