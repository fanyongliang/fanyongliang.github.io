2014年11月18日

  1. Spring MVC注解开发
    ·在springMVC配置文件中设置组件扫描为整个工程
    ·@Controller 标示当前类是控制器一个具体的实现
    ·@RequestMapping 在方法上指定该方法的访问路径，放在类上命名空间要组合指定的路径
  2. Spring MVC注解接收参数
    ·使用HttpServletRequest可以直接定义在参数的列表
    	@RequestMapping("/test1")
    	public String test1(HttpServletRequest request) {
    		String name = request.getParameter("name");
    		logger.info("test1 name:" + name);
    		return "home";
    	}
    ·在参数列表上直接定义要接收的参数名称，只要参数名称能匹配上的就能接收传过来的数据
    	@RequestMapping("/test2")
    	public String test2(String name, Date birthDate) {
    		logger.info("test2 birthDate:" + birthDate);
    		return "home";
    	}
    ·通过自定义实体对象接收参数
    	@RequestMapping("/test3")
    	public String test3(UserBean user) {
    		logger.info(user.toString());
    		return "home";
    	}
    ·接收数组
    	@RequestMapping("/test4")
    	public String test4(String[] name) {
    		for (String n : name) {
    			logger.info(n);
    		}
    		return "home";
    	}
  3. Spring MVC向页面绑定数据
    ·返回ModelAndView对象
    	@RequestMapping("/test6")
    	public ModelAndView test6() {
    		UserBean user = new UserBean();
    		user.setAge(17);
    		Map map = new HashMap();
    		map.put("u", user);
    		return new ModelAndView("home", map);
    	}
    ·通过Map参数
    	@RequestMapping("/test7")
    	public String test7(Map map) {
    		UserBean user = new UserBean();
    		user.setAge(17);
    		map.put("u", user);
    		return "home";
    	}
    ·通过Model参数
    	@RequestMapping("/test8")
    	public String test8(Model model){
    		UserBean user = new UserBean();
    		user.setAge(17);
    		model.addAttribute("u",user);
    		return "home";
    	}
  4. Spring MVC页面跳转
    ·跳绝对URL
    	@RequestMapping("/test9")
    	public String test9(){
    		return "redirect:http://www.gyyx.cn";
    	}
    ·跳Controller内方法
    @RequestMapping("/test10")
    	public String test9(){
    		return "redirect:test2";
    	}
    ·跳其他Controller内方法
    	@RequestMapping("/test11")
    	public String test9(){
    		return "redirect:Controller/test2";
    	}
